HTML Forms

Forms are wrapped in the tag: <form> insert form here </form>

Example:


    <form>
        First Name: <input type="text" name="firstname">
    </form>

Input Types

There are different input types for specific for items

  • <input type="text"> : input of type text
  • <input type="email"> : email formating
  • <input type="password"> : characters appear as ***
  • <input type="date"> : displays a calander to select dates
  • <input type="radio"> : selector button (multiple radios must be named the same but given different values)
  • <input type="checkbox"> : checkboxes (multiple checkboxes must be named the same but given different values)
  • <input type="reset"> : button to clear all filled out fields
  • <input type="submit" value="Register"> : button to submit form

Input Names

It is inportant to add names to each input item in the form! This way when submitted the answers can be stored/used.

Example:


    <form>
        First Name: <input type="text" name="firstname">
        Email: <input type="email" name="email">
        Password: <input type="password" name="password">
        Today's Date: <input type="date" name="date">
        Button: <input type="radio" name="button" value="one" > <input type="radio" name="button" value="two" >
        CheckBox: <input type="checkbox" name="checkbox" value="two" > <input type="checkbox" name="checkbox" value="two" >
        <input type="submit">
        <input type="reset">
    </form>

Usefull Fields

There are many handy fields to include in inputs to indicate how responses should look

  • required: ensures you may not submit form until section is complete
  • min="5": the minimum length must be at least 5 characters long
  • etc...
